# SafePlay Conductor Index

Welcome to the context registry for **SafePlay**, the high-performance offline-first crowd safety orchestrator for FIFA World Cup 2026 stadiums.

This directory manages the project context as a first-class version-controlled artifact.

## Context Artifacts

- **[Product Vision](product.md)**: Product definition, problem statement, user personas, core capabilities, and non-negotiables.
- **[Product Guidelines](product-guidelines.md)**: Communication standards, brand tone, terminology, and error conventions.
- **[Technical Stack](tech-stack.md)**: Programming languages, key libraries, dev tools, and target environments.
- **[Workflow & Quality Gates](workflow.md)**: Development practices, testing guidelines, git conventions, and safety standards.
- **[Tracks Registry](tracks.md)**: Active and completed engineering tasks and features.

## Core Directives

1. **Context Precedes Code**: Update these files before making major architectural changes.
2. **Offline Resilience**: Maintain the offline-first zero-latency execution model.
3. **Safety First**: Operator veto overrides, deterministic schema contracts, and strict latency bounds remain non-negotiable.
